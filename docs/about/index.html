About

This is an experiment where I’m trying to build a JAMstack site with Hugo and Azure Functions.

The source code is available at: https://github.com/hlaueriksson/jamstack-cms

The site is hosted by GitHub Pages at: https://hlaueriksson.github.io/jamstack-cms/

The jam recipes was taken from the Swedish site Tasteline.com:

  • Apple Jam
  • Lingonberry Jam
  • Strawberry Jam

Translations from Swedish to English was done with Google Translate.

The images was taken from Flickr:

  • Apples by 1sock
  • Lingonberries by Jyrki Kymäläinen
  • Strawberries by Fried Dough
